JavaScript common repos

WebRepository

ggondim/WebRepository

WebRepository is an engine for web storage using a provider abstraction, allowing you choose the best provider for your client, like IndexedDB, LocalStorage or SessionStorage.

WebVersioner

ggondim/WebVersioner

A JavaScript component to create and manage object versions.

WebReqr

ggondim/WebReqr

WebReqr is an engine for web requests using a provider abstraction, allowing you choose the best provider for your client, like jQuery's $.http, Angular's $http and any third-party provider.

WebPoolr

ggondim/WebPoolr

WebPoolr is an engine for web scokets using a provider abstraction, that automatically determines the best provider [WebSockets or AJAX requests].
